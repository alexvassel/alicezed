АКВАФОН - первый мобильный оператор Абхазии

Разработка

Техническая поддержка, исправление ошибок и доработка некоторых разделов сайта оператора мобильной связи ЗАО "АКВАФОН-GSM", дочерней компании "МЕГАФОН" в республике Абхазия. Крупный сайт с задачами различного уровня сложности. Например, была поставлена задача сделать всплывающую таблицу с выбранными тарифами для последующего их сравнения. Была продумана схема добавления тарифов по клику на галочку "Добавить тариф к сравнению" в бокую панель. После добавления по клику на кнопку "Сравнить" открывается таблица с характеристиками. Большое количество JavaScript, работа с массивами, переменными, cookies, Ajax. Проект реализован с использованием языка XSLT.
